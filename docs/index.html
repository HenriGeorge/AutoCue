How to use AutoCue

  1. Export from Rekordbox: File → Export Collection in rekordbox format → save as rekordbox.xml
  2. Drop the XML above. Optionally drop your audio files too to enable playback and artwork.
  3. Adjust settings and preview cue placements. Click any cue badge to audition it.
  4. Download the updated XML, then in Rekordbox: File → Import Library.

Everything runs locally in your browser — no files are ever uploaded anywhere.

Use the Python CLI instead ↓

The CLI reads directly from your Rekordbox database and uses phrase detection (Intro, Chorus, Outro) automatically — no ANLZ files needed.

pip install autocue

autocue --library                    # all analyzed tracks
autocue --track "Song Title"
autocue --track-id 42 --dry-run
autocue --library --playlist "My Set"
