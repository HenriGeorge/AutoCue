How to use AutoCue

  1. Export from Rekordbox: Open Rekordbox → File → Export Collection in rekordbox format → save as rekordbox.xml
  2. Drop the file into the zone above and adjust the cue interval to taste.
  3. Download the updated XML file.
  4. Import back: In Rekordbox → File → Import Library → select your downloaded file. Rekordbox will add the hot cues to your tracks.

AutoCue places hot cues at equal bar intervals using the BPM and beat-grid data already in your Rekordbox XML. Everything runs locally in your browser — no files are ever uploaded anywhere.
