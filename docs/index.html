How to use AutoCue (web mode)

  1. Export from Rekordbox: File → Export Collection in rekordbox format → save as rekordbox.xml.
  2. Drop the XML above. Optionally also drop your audio files (or the whole analysis folder) to enable playback, artwork, and phrase analysis.
  3. Choose analysis mode:
    • Bar intervals — places cues every N bars using the BPM grid. Works for every analyzed track.
    • ✨ Phrase analysis — uses phrase data from the dropped ANLZ files (Intro, Verse, Chorus, Outro). Drop your Rekordbox analysis folder to enable this.
  4. Adjust settings: bars interval, starting bar, max hot cues per track, and optional memory cue at bar 1. Hover any option for a tooltip explaining what it does.
  5. Preview cues: click any cue badge on a track card to audition the position. The timeline at the bottom of each card shows all cue positions at a glance.
  6. Download the updated XML, then import it back into Rekordbox: File → Import Library → select the file.

Everything runs locally in your browser — no files are ever uploaded anywhere.

Local server mode — workflow

  1. Close Rekordbox before applying, deleting, or coloring tracks. AutoCue writes directly to the Rekordbox database (master.db) — it must be closed first. AutoCue warns you if it detects Rekordbox is still running.
  2. Filter & browse your library:
    • Use the search bar to filter by title or artist.
    • Use the playlist dropdown to view a specific playlist.
    • Filter by Phrase only, Rating, Plays, Last Played, or My Tag.
    • Sort by Title, Artist, Album, BPM, Key, Rating, or Plays — click the same button again to reverse order.
    • In Album view tracks are grouped under collapsible album headers.
  3. Choose analysis mode:
    • Bar intervals — places cues every N bars using the BPM grid. Works for every analyzed track.
    • ✨ Phrase analysis — uses Rekordbox's built-in phrase detection (Intro, Verse, Chorus, Outro, etc.). Falls back to bar intervals if a track has no phrase data.
  4. Configure cue settings (hover each option for a detailed tooltip):
    • Bars interval / Start bar / Max hot cues — control bar-interval placement.
    • Memory cue at bar 1 — also writes a CDJ Auto Cue so your CDJ loads to the right position.
    • Fill-beat cues — phrase mode only; adds cues at the fill sections before drops.
    • Skip tracks with existing cues — leaves manually-cued tracks untouched.
  5. Select tracks (optional): check individual track cards to scope operations to a subset. Select all selects every visible (filtered) track. Apply, Delete, and Color by BPM all respect the current selection.
  6. Preview cues (optional): click Preview cues to generate positions without writing. Each card shows a secondary preview timeline. Click Apply when ready.
  7. Apply to Rekordbox: click the green button. AutoCue backs up master.db first, then streams progress as each track is processed. Use ↩ Undo last apply immediately afterwards to restore the backup if needed.
  8. Color tracks by BPM: assigns a Rekordbox track color by BPM range — Aqua (<90), Green (90–114), Blue (115–124), Orange (125–134), Red (135–149), Pink (≥150). Enable Skip colored to leave already-colored tracks untouched.
  9. Delete all hot cues: click Delete all cues → confirm. Removes every hot cue (slots A–H) from all visible tracks, including manually placed ones. A backup is made first.
  10. Restore a backup: click Restore backup to pick from previous database snapshots saved to ~/.autocue/backups/.
  11. Listen to tracks: hover over the album art and click play. AutoCue loads the audio directly from disk — no file dropping needed. You can also drop an audio file onto the Load audio button on any card.

Backups are saved to ~/.autocue/backups/ before every write operation. The server only accepts connections from localhost — your database never leaves your machine.

Use the Python CLI or local server instead ↓

The Python package provides both a CLI and a local server that writes directly to your Rekordbox database — no XML export/import needed.

pip install autocue

# Local server (this UI) — reads & writes the Rekordbox DB directly
autocue serve                        # opens browser automatically
autocue serve --no-browser           # start without opening browser
autocue serve --port 8080            # use a different port

# CLI — generates a new rekordbox.xml with cues
autocue --library                    # all analyzed tracks
autocue --library --overwrite        # re-generate for all (overwrite existing)
autocue --library --dry-run          # preview without writing
autocue --track "Song Title"
autocue --track-id 42 --dry-run
autocue --library --playlist "My Set"
autocue --library --mode phrase      # force phrase mode
autocue --library --mode bar --bars-interval 8 --start-bar 1

DJ Mixing Guide
▼

Energy Modes

  • Build (ascending) — Classic warm-up → peak → send-off. BPM creeps 1–2 per transition. Leave your biggest tracks for the final third.
  • Flat — Sustained peak energy. Works for festival prime-time slots. Vary texture (vocal/instrumental, key changes) every 3–4 tracks to prevent crowd fatigue.
  • Drop (descending) — Energy descends — ideal for a closing set or handing off to the next DJ at a lower intensity.

Structuring the arc

  • Think in blocks of 3–4 tracks at similar energy rather than increasing every transition. Small plateau dips make the next peak hit harder.
  • The reset: Drop energy once per set, maximum. Deliberately rebuild from a lower floor — more than once becomes exhausting.
  • Closing: Ease BPM down 2–3 per transition; shift from driving to groover tracks; move from minor keys toward major for a brighter feel.
  • Young crowds (18–25) need energy variation every 5–10 min. Mature crowds (30+) sustain energy for 15–20 min before needing a texture change.

BPM jump size

  • ±1–4 BPM Organic, within-genre step. Standard.
  • ±5–10 BPM Genre-stepping zone. Use a filter sweep, breakdown, or loop hold to mask the change.
  • ±10+ BPM Deliberate statement. Drop outgoing volume, bring the new track in clean.

AutoCue default (BPM step max 8%)

8% ≈ the boundary between a "moderate" and "large" jump. Tracks at the gate edge get a 15% similarity score penalty. Tighten to 4–5% for smoother genre-consistent sets; widen only if you plan to use filter/EQ techniques.

Pitch & Key Lock

  • Safe pitch shift: ±2 semitones (≈±6% tempo) before audio artifacts become audible to trained ears.
  • Every 3% tempo change = 1 semitone pitch shift when Key Lock is off.
  • Key Lock artifacts appear beyond ±5% — especially on reverb tails and transients.
  • Half/double time: bridge any gap — 160 → 80 or 80 → 160 without pitch issues.

The 4 core Camelot moves

8A → 8A
Always safe Same key — perfect harmonic match
8A → 7A / 9A
Safe ±1 position — one-note scale change, imperceptible to most listeners
8A → 8B
Safe A↔B flip — relative minor/major, same harmonic center. A→B lifts the mood.
8A → 3A
Intentional +7 positions — perfect fifth up, 6/7 shared notes. Strong crowd energy lift.
8A → 10A
Intentional +2 positions — whole-tone up. Noticeable upward shift, use deliberately.

In practice

  • Spend ~80% of your set on same-key and ±1 moves; reserve +2 and +7 for intentional climactic moments.
  • Harmonic mixing is a probability filter, not a guarantee. If it sounds wrong in headphones, it's wrong — regardless of matching codes.
  • Camelot numbers wrap: 12+1 = 1 (so 12A → 1A is a safe ±1 move).

Bassline swap technique

  • Bring incoming track in with bass/low EQ cut to zero.
  • When the incoming bassline arrives, restore it while simultaneously cutting the outgoing bass.
  • Never run two kick drums for extended periods — it sounds muddy, not powerful.

EQ tricks

  • Vocal clash fix: cut mid EQ on outgoing track as incoming vocals enter (both occupy 1–4 kHz).
  • Bass cut for lift: a 1–2 bar bass cut at a phrase end gives momentary weightlessness; 8 bars builds real tension. Use sparingly.
  • ½–1 beat silence before a drop measurably increases perceived impact.
  • Low-pass filter intro: loop the incoming intro with filter all the way left, then open gradually — creates tension and emergence.

Phrasing

  • The 16-bar phrase is the fundamental mixing window in electronic music. Start blends at the beginning of a phrase.
  • Hot Cue A (Mix In) marks the first useful beat — use it to jump past silence and dead air.
  • Hot Cue B (Outro) marks your mix-out point — the start of the outro, not the end of the track.

Positive signals — keep going

  • Bodies moving toward the speakers / booth
  • Hands raised, synchronized movement
  • Best dancers maintaining eye contact with the booth
  • <15% phone use during breakdowns
  • People rushing back from the bar after a build

Warning signals — adjust immediately

  • >30% of the crowd drifting toward the bar
  • Arms crossed, backs turned, conversations overpowering music
  • <15% moving during a breakdown (energy fatigue, not anticipation)
  • Drinks being picked up in large numbers (fatigue signal)

Managing peaks

  • The strongest peaks come from restraint — holding back your biggest track until the right moment signals confidence.
  • A well-placed groover after a peak makes the following drop land harder.
  • Keep 3–5 emergency tracks ready — pop-dance crossovers that work for any crowd and any mood.
  • Quick cuts (0–8 beats) reset energy; filter sweeps ease transitions without killing momentum.
