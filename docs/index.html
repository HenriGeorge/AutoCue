How to use AutoCue

  1. Export from Rekordbox: File → Export Collection in rekordbox format → save as rekordbox.xml
  2. Drop the XML above. Optionally drop your audio files too to enable playback and artwork.
  3. Adjust settings and preview cue placements. Click any cue badge to audition it.
  4. Download the updated XML, then in Rekordbox: File → Import Library.

Everything runs locally in your browser — no files are ever uploaded anywhere.

Local server mode — workflow

  1. Close Rekordbox before applying or deleting cues. The Rekordbox database is locked while the app is open. AutoCue will warn you if it detects Rekordbox is running.
  2. Choose analysis mode:
    • Bar intervals — places cues every N bars using the BPM grid. Works for every analyzed track.
    • ✨ Phrase analysis — uses Rekordbox's phrase detection (Intro, Verse, Chorus, Outro). Falls back to bar intervals if a track has no phrase data.
  3. Configure cue settings: set how many bars apart each cue lands, which bar to start from, and the maximum number of cues per track.
  4. Browse your library: sort by Title, Artist, Album, or BPM (click the same button again to reverse order). In Album view tracks are grouped under collapsible album headers — click an album to expand it. Each track card previews the cues that will be written.
  5. Skip existing cues: check Skip tracks that already have hot cues to leave tracks untouched. Uncheck it to overwrite all tracks.
  6. Apply to Rekordbox: click the green button. AutoCue backs up master.db first, then writes the cues directly. The footer shows how many tracks and cues will be written before you confirm.
  7. Delete all hot cues: click Delete all cues (bottom left) → confirm in the inline bar. This removes every hot cue (slots A–H) from all tracks, including manually placed ones. A backup is made first.

Backups are saved to ~/.local/share/autocue/backups/ (macOS: ~/Library/Application Support/autocue/backups/) before every write. The server only accepts connections from localhost — your database never leaves your machine.

Use the Python CLI instead ↓

The CLI reads directly from your Rekordbox database and uses phrase detection (Intro, Chorus, Outro) automatically — no ANLZ files needed.

pip install autocue

autocue --library                    # all analyzed tracks
autocue --track "Song Title"
autocue --track-id 42 --dry-run
autocue --library --playlist "My Set"
